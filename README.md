## Descrição

Projeto destinada a criar uma aplicação web de cardápios digitais.

<div id='como-rodar'/>

## Como rodar esse projeto

Caso opte por utilizar em servidor local, será necessário utilizar oos seguintes comandos em sequência:

```bash
$ npm install

$ npm run dev
```
